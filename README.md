# Personal Task Manager

This is a simple console based **Task Manager application** built using **Python and Object Oriented Programming (OOP)**.

The program allows users to manage daily tasks through a menu-driven system.

## Features

* Add Task
* Remove Task
* Show All Tasks
* Update Task
* Mark Task as Completed
* Show Completed Tasks
* Show Pending Tasks

Tasks are stored using a **list**, and each task has a status (Pending or Completed).

## How to Run

Run the following command in the terminal:

python task_manager.py
